# LanOS -- 梳理linux0.12知识点系列

	这个工程的目标是梳理《Linux源码剖析》的知识点并从0.12版本的源码中抠出部分代码构建一个
	能用现代linux发行版方便编译的迷你系统，从而验证各个知识点
